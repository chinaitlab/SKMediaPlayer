# SKMediaPlayer
Joyway Media Player

久为影音

V1.0版功能：

    1、支持双击全屏，ctrl+enter为快捷全屏  
    2、空格键为暂停
    3、左右键为快进后退
    4、上下键为音量控制
    5、多种皮肤切换

V2.0版功能：

    1、增加全屏时，鼠标移至下部出现播放器工具栏
    2、增加右键菜单功能
    3、增加音轨选择功能，主要用于声音的多语切换
    4、增加显示比例和放大，缩小功能
    5、增加全屏时，系统时间，视频时间进度和音量的显示，可人工去除
    6、优化视音代码，减小软件体积 
  
V2.0.1-2.0.4版功能：

    1、添加快照功能
    2、修正音轨描述乱码问题
    3、修正编码崩溃问题
    4、修正播放完毕崩溃问题
    5、修正跳转崩溃问题

V2.0.5版功能：

    1、添加视频时间记录，续播功能
    2、添加影音列表功能
	
V2.0.6版功能：

    1、添加影音列表中增加，删除列表项功能
